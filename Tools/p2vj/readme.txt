P2VJ は、PNG や Jpeg といったラスタ画像をSVGに変換してくれるソフトウェアです。
P2VJ.jar をダブルクリックしたらたいていの場合は起動しますが、runp2vj.bat をダブルクリックして起動すると、コンソールも開くし、エラーメッセージが見えるので、後者をお勧めします。
ウインドウの中に画像ファイルをドラッグアンドドロップすれば SVG 化された状態で同じフォルダにセーブされます。

設定ファイルが p2vjsettings.ini として保存されるので、同じ名前のファイルを同じフォルダにおかないでください。
0.9 から、CUI で実行することが出来るようになりました。


＊チェックボックスについて
posterize･･･ポスタリゼーション（色の平滑化）をかけます。写真や、ぼかしを使っている絵を変換しようとする場合ONにしておいてください。jpeg画像の場合もONにしておくことをお勧めします。
remove anti-Aliase･･･完全に2値化された絵（MSペイントなどで描かれた絵）を変換する場合以外はONにしておいてください。
binary alpha･･･アルファチャンネル要素が 64 未満の場合完全透明、以上の場合は完全不透明にします。
makepng･･･ベクタ化した後PNG画像を作ります。
makepdf･･･ベクタ化した画像の PDF ファイルを作ります。コンビニとかで高解像度画像が（たぶん）そのまま印刷できるよ！


＊CUI モードについて
コンソールから起動する際、-i <ファイル名> というオプションを入れると、GUI を使用せず実行することが出来ます。
使い方

java -jar P2VJ.jar -i <inputfilename> <options...>

-i <変換したい画像ファイル名> 
　必須です。これが無いと、GUI が起動します。

-o <SVG 化した画像を保存する際のファイル名>
　これが無い場合は標準出力に内容が出力されます。

-png <SVG 化した画像をさらに PNG 化したい時、保存するファイル名>

-png_scale <PNG 化した際の画像の倍率>
　PNG ファイルの大きさを変えたい時。％表記ではありません。

-pdf <PDF 化した画像を保存する際のファイル名>
　CUI モードでは 1 枚ずつしか保存できません。

-posterize <色数>
　減色処理を行います。またその際の処理後の色数（厳密ではありません）を指定します。
　写真等の場合、これで 256 くらいを指定しないとめちゃ遅いです。

-remove_aa <TRUE or FALSE>
　TRUE を与えるとアンチエイリアスを消去します。
　デフォルトで TRUE です。小さい画像なんかは FALSE を指定した方がいいかもしれません。

-h
ヘルプの表示
（大したことは出ません）

既知の不具合
windows 7 64bitにおいて、Javaのパスがデフォルトから変更されており、動かないという報告があります。
その場合、"runp2vj.bat"の中身を以下の様に書き換えて保存し、"runp2vj.bat"をダブルクリックしてみてください。

"C:\Program Files (x86)\Java\jre7\bin\javaw.exe" -jar P2VJ.jar
pause


ここの "jre7" は、Java のプログラムが保存されているフォルダであり、"7" となっていることからも分かるように Java7 特異なものです。違うバージョンの Java を使っている場合、変更する必要があります。
違うバージョンの Java を使っている人は、javaw.exe と打ち込んで javaw.exe がどこにあるか調べ、そのフォルダへのパスを設定してください。
windows 10 での動作は確認していません。。。
