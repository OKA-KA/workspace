-------------------------------------------------------------------------------
 Div - ファイル分割／結合ユーティリティー
-------------------------------------------------------------------------------

●はじめに
　普通にファイルサイズを決めて分割するソフトは沢山ありますが、テキストファイル
の行数を決めて分割するソフトはなかなか無いようで、インターネットで探すのが面倒
になったので作りました。もともとは、偏微分方程式を数値計算で解いた結果できてし
まう巨大な（1GBを超えることもあります）テキストファイルを読みこんで、必要なとこ
ろだけ取り出したりする際に、時間を短縮するためにファイルを細切れにすると良いと
いうことから作成しましたが、どうも普通にファイルサイズを決めて分割する方が頻繁
に使われるようなので、この機能もつけました。


●動作環境
　Microsoft Windows XP SP2 で動作確認しました。


●特徴
1.従来のファイルサイズ指定によるバイナリ分割に加え、テキストファイルを行数指定
　での分割が可能
2.バイナリ分割モードでは、FDDやMOなどの固定されたサイズ、また任意のサイズでの
　分割が可能
3.テキスト分割モードでは、固定された単位分割行数として1,000〜1,000,000行の選択
　ができるほか、任意の単位分割行数も指定可能
4.分割作業の完了後、結合用のバッチファイルを自動生成
5.バッチファイルを本プログラムで読み込み結合作業をすることにより、ファイルの属
　性とタイムスタンプの復元が可能
6.分割処理中、終了までのおおよその残り時間を表示
7.ドロップされたファイルの拡張子に応じ、バイナリ分割、テキスト分割、結合を自動
　的に設定
8.CRC値を調べることにより結合元ファイルの正当性を評価可能


●パッケージ内容
　Div110a.lzh には以下のファイルが含まれています。
　　Div     .EXE    Divのプログラム本体
　　Div     .HLP    Divのヘルプファイル
　　Div     .CNT    ヘルプの目次
　　ReadMe  .TXT    このファイル


●インストール方法
　Div.EXEとDiv.HLPとDiv.CNTを適当なディレクトリにコピーしてください。


●アンインストール方法
　Divに関するレジストリの内容を削除するため、Divが起動した状態から、メニュー中
の[ヘルプ⇒レジストリのクリアと終了]を選択してください。その後Div.EXEとDiv.HLP
とDiv.CNTを削除してください。また、Div.GIDやDiv.FTSが存在する場合はこれも削除し
てください。
