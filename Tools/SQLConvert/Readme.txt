１　SQLConvertとは

・SQLConvertは、SQL文(Select,Upadate,Insert)を人が見やすいように改行・スペースを
　自動的に入れて見やすくするツールです。


２　特徴

・SQL文をコピーしてSQLConvert画面の「実行」ボタンを押すだけでSQL文が
　綺麗に変換されます。貼り付けを行うと変換後のSQL文が貼り付きます。
・秀丸エディタでは、選択している文字列のみを変換することが出来ます。（マクロ登録時）
・ウィンドウを表示せずに変換することも出来ます。（タスク起動時）
・開発者向けに文字列をSQL文の前後に付加する機能もあります。


３　セットアップ方法

・圧縮ファイルを解凍後[SQLConvert.exe]を起動させると使用できます。
・秀丸エディタで使用する場合は、秀丸エディタのマクロに[SQLConvert.mac]を
　登録する必要があります。また、[SQLConvert.mac]ファイルを開いて[SQLConvert.exe]を
　配置したパスに変更する必要があります。
　※秀丸のマクロは、\を２つ書くことで１つとして認識します。
・[SQLConvert.exe]のショートカットを作成して/tを引数で渡すとタスク起動になります。
・[SQLConvert.exe]のショートカットを作成して/sを引数で渡すと即、SQL整形を行います。

※Microsoft .NET Framework Version 3.5以降がインストールされていないと
　正常に動作いたしません。インストールされていない方は、
　MicrosoftWindowsUpdate等でインストールしてください。


４　動作環境

　対応ＯＳ
　　Windows XP
　　Windows Vista
　　Windows 7
　　Windows Server 2003
　　Windows Server 2008
　　Windows Server 2008R2


５　使用方法

　１．整形したいSQL文をメモ帳等でコピーします。
　２．「SQLConvert.exe」をダブルクリックします。
　　　（初回のみ起動に時間がかかります。）
　３．「実行」ボタンを押します。
　４．メモ帳等に貼り付けます。
　　　（整形されたSQL文が貼り付きます。）
